# env

create my development environment.

## Provisioning

use MItame.
https://github.com/k0kubun/mitamae

```sh
wget https://github.com/k0kubun/mitamae/releases/download/v*.*.*/mitamae-* -O mitamae
chmod +x mitamae
./mitamae local -j itamae/vagrant.json itamae/base/*
```
